GETTING STARTED

npmではじめる

使い慣れたパッケージマネージャーで資料を作るか、手元のSlidev資料へテーマを追加します。

npm

新しい資料を作る

npm create @timdaik/nutmeg-slides@latest -- my-slides

既存資料へテーマを追加

npm install -D @timdaik/slidev-theme-nutmeg@latest

yarn

新しい資料を作る

yarn create @timdaik/nutmeg-slides my-slides

既存資料へテーマを追加

yarn add --dev @timdaik/slidev-theme-nutmeg@latest

pnpm

新しい資料を作る

pnpm create @timdaik/nutmeg-slides@latest my-slides

既存資料へテーマを追加

pnpm add --save-dev @timdaik/slidev-theme-nutmeg@latest

bun

新しい資料を作る

bun create @timdaik/nutmeg-slides@latest my-slides

既存資料へテーマを追加

bun add --dev @timdaik/slidev-theme-nutmeg@latest

deno

新しい資料を作る

deno run -A npm:@timdaik/create-nutmeg-slides@latest my-slides

既存資料へテーマを追加

deno add --dev npm:@timdaik/slidev-theme-nutmeg@latest

Denoでは、生成コマンドにパッケージの実行権限（-A）を渡します。
